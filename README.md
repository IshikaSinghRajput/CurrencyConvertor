# 💱 Currency Converter – Real-Time Exchange Rates

A simple and responsive **Currency Converter web app** built using **HTML, CSS, and Vanilla JavaScript**.  
Convert between multiple currencies with live exchange rates and a clean, user-friendly UI.

---

Demo : https://currencyconverterbyishika.netlify.app/

## 🎯 Features

- 💱 Convert between popular currencies (USD, INR, EUR, GBP, etc.)
- 🌐 Real-time exchange rates (via public API)
- 📱 Fully responsive UI (works on mobile & desktop)
- ⚡ Fast and lightweight (no frameworks)
- 🎨 Clean and minimal design
- 🔁 Swap currencies easily
- ❌ Input validation for better UX

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling & responsiveness  
- **JavaScript (Vanilla)** – Logic & API handling  
- **Public Exchange Rate API** – Live currency data
